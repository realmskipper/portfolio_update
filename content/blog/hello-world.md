# Hello World

This is the first post on the new self-hosted blog. The hybrid setup lets me write posts here in markdown **and** link out to my existing Medium articles.

## How it works

- Drop a `.md` file in the `content/blog/` folder
- Add frontmatter with title, date, description, and tags
- It shows up on the blog index automatically

Simple as that.
